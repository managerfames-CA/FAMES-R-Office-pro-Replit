# Audit Firm Management App — Complete Phase 6

Complete standalone React + TypeScript + Vite application for an audit and professional-services firm. Phase 6 integrates and hardens all functionality delivered in Phases 1–5. It remains a local, single-browser Version 1 application using namespaced `localStorage` through repository interfaces and business services.

## Technology

- React 19
- TypeScript 6
- Vite 8
- React Router
- localStorage repository adapter
- Vitest and Testing Library

No Python, Supabase, login, cloud storage, database server, API key, Docker, or environment secret is required.

## Local Run on Windows

1. Extract the ZIP.
2. Open PowerShell or Command Prompt inside the extracted project folder.
3. Run:

```powershell
npm install
npm run dev
```

4. Open the local URL printed by Vite, normally `http://localhost:5173`.

## Verification Commands

```powershell
npm run typecheck
npm run test:run
npm run build
```

The production files generated by `npm run build` appear in `dist/`. The distributed ZIP intentionally excludes `dist` and `node_modules`.

## Architecture

```text
React UI
→ Business Services
→ Repository Interfaces
→ localStorage Repository Adapter
```

Pages and components do not directly access operational localStorage records. Phase 6 recovery functions are centralized in `Phase6Service` and use the configured storage gateway.

## Backup Compatibility

Current backup schema: **7.0**.

Older schemas are rejected with a precise migration message rather than being silently imported. Before replacing data, export a current full JSON backup.

## Important Limitations

- Data is local to the current browser profile/device.
- Activity history is operational, not tamper-proof.
- No actual document or receipt bytes are stored.
- No login, permissions backend, cloud sync, scheduled notification, or client portal is included.
- Vite reports a non-blocking large JavaScript chunk warning; build and runtime verification pass.

See `PHASE6_COMPLETION_REPORT.md` and the `reports/` directory for full evidence.
